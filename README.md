# glym

Tiny glimmers for your terminal. A TUI library for Zig.

`glym` is a Model-View-Update framework for building terminal user interfaces
in Zig. Part of the [glymmi](https://github.com/glymmi) ecosystem.

> Status: very early. APIs will change.

## Requirements

- Zig 0.15.x stable
- Linux, macOS, or Windows

## Quick start

```sh
git clone https://github.com/glymmi/glym
cd glym
zig build test
```
